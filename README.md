# saas_exit_playbook_tracker

追踪和管理SaaS快速退出策略的工具

## 功能
- 行业筛选与评估（legacy industry scoring）
- 潜在收购方关系管理（design partner CRM）
- 定期沟通记录与提醒（rapport building tracker）
- 收购机会识别与匹配（opportunity detection）
- 交易进度管理（deal pipeline）
- 关键指标仪表板（legitimacy metrics: funding, team size, partnerships）

## 安装
```bash
pip install -r requirements.txt
python src/main.py --help
```

## 技术栈
Python, Flask/FastAPI, SQLite/PostgreSQL, pandas
